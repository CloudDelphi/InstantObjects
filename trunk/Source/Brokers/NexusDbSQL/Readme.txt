NexusDbSQL Brokers for InstantObjects
Steven Mitchell - 24 Apr 2005

To use these brokers with NexusDb V1 use a compiler conditional define of "Nx1" in the runtime package options.

To use these brokers with NexusDb V2 remove the compiler conditional define of "Nx1" in the runtime package options.

The current default is to compile for NexusDb V1.

Note: Minimal testing has been done with NexusDb V2 as this version has not been released yet.
