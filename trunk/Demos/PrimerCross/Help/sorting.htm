Sorting

Objects retrieved via a selector can be sorted by the underlying database by specifying one or more persistent attributes of the object or related objects with the ORDER BY clause. In addition, objects exposed through an exposer or a selector can be sorted in memory by adding a custom sort method to the OnCompare event.

InstantObjects Primer demonstrates both sort methods via the Sort option available on the toolbar of the Contacts view. A custom sort event is also used by the exposer EmployeeExposer in TCompanyEditForm to order the employees by their name.
