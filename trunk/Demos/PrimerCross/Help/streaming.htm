Streaming

All business objects created with InstantObjects can be written to and read from any standard VCL stream. The structure of the stream can be either binary or XML.

An example of streaming objects to an XML file is supplied by the Export to XML option on the toolbar of the Contacts view. This option can also be selected by right-clicking on a contact in the grid. When executed, the export function writes the currently selected contact to an XML file and launches your default XML viewer with the file. This would normally be your Internet browser.
